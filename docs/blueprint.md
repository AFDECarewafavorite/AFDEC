# **App Name**: AFDEC Online

## Core Features:

- Multi-Step Booking Form: A step-by-step wizard to guide customers through the chicken booking process, auto-saving progress at each step.
- Bird Type Selection: Allows customers to select the type of bird (Day-old chicks, Grower chickens, Mature chickens) with associated images and descriptions displayed automatically.
- Quantity Selection with Live Preview: Customers select the desired quantity of chickens using plus/minus buttons, with a live preview displaying the corresponding number of chicken icons.
- Customer Details Capture: Collects necessary customer information (Full name, Phone number, Location, Referral code) during the booking process.
- Booking Summary & Confirmation: Presents a summary of the booking details (bird type, quantity, booking fee) and a clear notice about the manager contacting them.
- Payment Integration: Enables customers to pay the booking fee via Paystack, generating a booking ID and saving the booking with 'pending' status upon successful payment. Confirmation message (screen + WhatsApp) after success
- Agent Referral System: Automatically assigns bookings to agents based on referral codes and manages commission calculations.
- Admin Dashboard: Displays bookings with customer details, bird info, booking status, click-to-call, and the ability to update status, add notes, and manage agents/commissions.

## Style Guidelines:

- Primary color: Gold (#FFD700) reflecting the poultry and premium service.
- Background color: Dark slate gray (#37474F), a desaturated version of the primary hue, providing a neutral and professional backdrop.
- Accent color: Forest green (#228B22) analogous to gold but contrasting for highlights and calls to action.
- Body font: 'PT Sans', a humanist sans-serif, for clear, readable text.
- Headline font: 'Space Grotesk', a bold sans-serif font with a futuristic touch
- Use clear, simple icons representing bird types, quantities, and booking status.
- Mobile-first design with large, touch-friendly buttons for ease of use on Nigerian mobile phones.
- Subtle animations on booking steps and data updates to enhance user engagement.
